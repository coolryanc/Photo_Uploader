### Edit photo
> Implement by HTML5 Canvas.
- - [x] Crop image, use vanilla js without any node package.

- - [ ] Filter presets. Reference: [CSSCO](http://www.cssco.co/).

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn start
```
